# Rhexia (Rhex-I-A)

Rhexia or Rhex Intelligence Artificielle, is a interpreted language written in C#.

Code Sample:
```
var name = "Rhexis";
var age = 30;
var daysOld = 10_950;

function greet(name)
{
    print("Hello, " + name);
}

greet(name);

print("Age is: " + age);
print("Name is: " + name);

if (age == 30)
{
    print("You are 30 years old!");
}
else
{
    print(age + " is " + age * 365 + " days old");
}

for(i in range(age))
{
    print(i);
}
```
